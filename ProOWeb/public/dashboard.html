Le socle est pret. Vous pouvez maintenant modeliser l'organisation, les flux BPMN et les donnees, puis faire evoluer les generateurs selon vos publications/recherches.

Commandes npm rapides

  • npm run compile - build backend + frontend
  • npm test - tests backend + frontend
  • npm run start:dev - docker compose profile dev
  • npm run start:demo - docker compose profile demo
  • npm run start:test - docker compose profile test
  • npm run start:preprod - docker compose profile preprod
  • npm run start:prod - docker compose profile prod
